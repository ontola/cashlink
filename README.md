# Cash-link

Service for sending cash to Payment Pointers through sharable links!

Created with help from [this tutorial](https://medium.com/swlh/create-an-api-rest-with-nestjs-1954723e8234).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
